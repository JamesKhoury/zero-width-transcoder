Zero Width Transcoder





Erroar

Encoding
Place the text you want to hide the message in into the Input field, then place the text you want to hide into the Secret field, and click Encode.
The original text should appear in the output field, but it will have the secret text hidden inside it.
Decoding
Place text that has the hidden text inside it (for example, copy, then paste the text from the result field from the encoding step) into the Input field.
Click on the Decode button, and a copy of the hidden text will appear in the Result field.
